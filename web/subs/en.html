Cheat sheet: tags in cue text

Commands are written in braces and affect the following text fragment. Line break: \N.

Size and weight

  • {\fs80} — 80 px size
  • {\b1} / {\i1} — bold / italic
  • {\b0} — return to regular weight

Cue color

  • {\c&H00FFFFFF&} — primary color (BGR format)
  • {\1c&H0000FF&} — fill only, {\3c…} outline
  • White #FFFFFF → &H00FFFFFF&, red #FF0000 → &H000000FF&

Karaoke

  • {\k50} — syllable highlight for ~0.5s (50 centiseconds)
  • {\kf50} — smooth syllable fill
  • Example: {\k40}Hel{\k40}lo, {\k60}world!
  • For karaoke set SecondaryColour in the style: it is the color of the unrevealed part.

Position move

  • In the cue form: ΔX, ΔY, start and duration (ms).
  • Start is the style position; finish is the ΔX/ΔY offset.
  • In ASS: {\move(x1,y1,x2,y2,200,700)} moves from 200 to 700 ms.

Full example

{\fs90\c&H00FFFF&}{\k45}Hel{\k45}lo {\k60}text

Burn subtitles

Locally, you can do it with this command: 

ffmpeg -i input.mp4 -vf "subtitles=subtitles.ass:fontsdir=./web/subs/fonts" -c:v libx264 -crf 18 -preset veryfast -c:a copy output.mp4
